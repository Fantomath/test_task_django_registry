Django Registry Test Task

Данное приложение реализует простой реестр на Django с возможностью добавления и удаления записей.

Приложение содержит модель реестра со следующими полями: ФИО, дата рождения, сумма кредита и срок кредита в месяцах. Поле ID используется как первичный ключ и не отображается на странице.

На странице по адресу /registry/ реализованы:

Добавление новой записи через форму.
Отображение всех записей в виде таблицы.
Удаление записи через кнопку «Удалить».

Стек проекта: Python 3.14, Django 6.0, SQLite.
